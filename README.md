RPSG (Rock Paper Scissors Game)
===============================

Requirements
------------

### For Windows

+ Ruby
- RubyGems

### For Mac

+ Ruby
- RubyGems
* RVM

How To Use
----------

### Windows

1. Open `cmd` as non-admin
2. type the following: `gem install rpsg`
3. run the following: `rps`

### Mac

1. Open `Terminal`
2. type the following: `gem install rpsg`
3. run the following: `rps`
