How To Use
==========

## Requirements

+ Ruby
- RubyGems

For Windows
-----------

1. Open `cmd` as non-admin
2. type the following: `gem install rpsg`
3. run the following: `rps`

For Mac
-------

1. Open `Terminal`
2. type the following: `gem install rpsg`
3. run the following: `rps`

----

RubyGems page: [https://rubygems.org/gems/rpsg](https://rubygems.org/gems/rpsg)
