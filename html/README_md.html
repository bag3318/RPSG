    How To Use¶ ↑

    Windows¶ ↑

    1. Open cmd as non-admin * cmd is the Command Prompt

    2. type the following: gem install rpsg

    3. run the following: rpsg

    Mac¶ ↑

    1. Open Terminal

    2. type the following: gem install rpsg

    3. run the following: rpsg
